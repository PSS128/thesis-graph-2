Add User Authentication with JWT (Simple Auth)                                                                │ 
     │                                                                                                               │ 
     │ Overview                                                                                                      │ 
     │                                                                                                               │ 
     │ Implement user login/logout system where each user only sees their own projects and data. Use JWT tokens for  │ 
     │ session management.                                                                                           │ 
     │                                                                                                               │ 
     │ Configuration Chosen                                                                                          │ 
     │                                                                                                               │ 
     │ - Auth method: Simple JWT (email + password)                                                                  │ 
     │ - Existing data: Assign to default user                                                                       │ 
     │ - User info: Email + password only                                                                            │ 
     │ - Password reset: Not needed                                                                                  │ 
     │ - Frontend: Provide React/Next.js components                                                                  │ 
     │                                                                                                               │ 
     │ ---                                                                                                           │ 
     │ Backend Changes                                                                                               │ 
     │                                                                                                               │ 
     │ 1. Install Dependencies                                                                                       │ 
     │                                                                                                               │ 
     │ Add to backend/requirements.txt:                                                                              │ 
     │ - python-jose[cryptography] (JWT tokens)                                                                      │ 
     │ - passlib[bcrypt] (password hashing)                                                                          │ 
     │ - python-multipart (form data)                                                                                │ 
     │                                                                                                               │ 
     │ 2. Create User Model (backend/app/models/store.py)                                                            │ 
     │                                                                                                               │ 
     │ - Add User table: id, email (unique), hashed_password, created_at, is_active                                  │ 
     │ - Update Project table: add user_id foreign key                                                               │ 
     │ - Update Feedback table: add user_id (optional, for analytics)                                                │ 
     │                                                                                                               │ 
     │ 3. Create Auth Dependencies (backend/app/dependencies/auth.py - NEW)                                          │ 
     │                                                                                                               │ 
     │ - hash_password() - bcrypt hashing                                                                            │ 
     │ - verify_password() - check password                                                                          │ 
     │ - create_access_token() - generate JWT                                                                        │ 
     │ - get_current_user() - verify token, return User                                                              │ 
     │ - Secret key configuration from environment                                                                   │ 
     │                                                                                                               │ 
     │ 4. Create Auth Router (backend/app/routers/auth.py - NEW)                                                     │
     │                                                                                                               │ 
     │ - POST /auth/register - Create account (email, password)                                                      │ 
     │ - POST /auth/login - Login (returns JWT token)                                                                │ 
     │ - GET /auth/me - Get current user info                                                                        │ 
     │ - POST /auth/logout - Optional (client-side token deletion)                                                   │ 
     │                                                                                                               │ 
     │ 5. Update Projects Router (backend/app/routers/projects.py)                                                   │ 
     │                                                                                                               │ 
     │ - Add current_user: User = Depends(get_current_user) to all endpoints                                         │ 
     │ - Filter queries by user_id:                                                                                  │ 
     │   - list_projects() - only user's projects                                                                    │ 
     │   - create_project() - set user_id automatically                                                              │ 
     │   - load_project() - verify ownership                                                                         │ 
     │   - save_project() - verify ownership                                                                         │ 
     │   - delete_project() - verify ownership                                                                       │ 
     │                                                                                                               │ 
     │ 6. Update Main App (backend/app/main.py)                                                                      │ 
     │                                                                                                               │ 
     │ - Register auth router                                                                                        │ 
     │ - Update CORS to allow Authorization header                                                                   │ 
     │                                                                                                               │ 
     │ 7. Database Migration Script (backend/migrations/add_users.py - NEW)                                          │
